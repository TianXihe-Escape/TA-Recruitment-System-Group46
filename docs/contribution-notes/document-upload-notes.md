# Document Upload Notes

## Purpose

This note explains how CV and supporting-document references are handled in the system.

## Why document references matter

TA applicants need to provide evidence that supports their application. The system stores local document references so these materials can be reviewed later by MOs.

## CV handling

Each applicant profile can include a CV path. This helps:

- preserve a local reference to the applicant CV
- allow the document to be opened during review
- support demonstration of file-related workflow features

## Supporting-document handling

Applicants can also provide supporting documents such as:

- certificates
- competition evidence
- additional academic or technical proof

## Validation role

Document paths are checked so unsupported file types can be rejected before invalid references are saved.

## Practical value

This approach keeps the workflow simple while still demonstrating realistic recruitment supporting materials in a local desktop application.
