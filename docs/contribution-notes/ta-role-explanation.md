# TA Role Explanation

## Purpose

This note explains the responsibilities and workflow of the TA applicant role in the system.

## Main responsibilities

The TA role is responsible for preparing an application-ready profile and engaging with the recruitment process from the applicant side.

## Typical TA actions

TA applicants can:

- create an account
- log in
- complete a profile
- add skills and academic details
- upload a CV
- browse available jobs
- submit applications
- monitor application status
- read notifications and messages

## Why this role matters

The TA role is central to the recruitment flow because applicant data, uploaded documents, and application records all begin from this side of the system.

## Relationship to other roles

TA applicants interact with jobs created by MOs and receive outcomes that can later be monitored by Admin users.
