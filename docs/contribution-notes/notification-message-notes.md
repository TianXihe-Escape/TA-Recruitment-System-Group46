# Notification and Message Notes

## Purpose

This note explains the difference between notifications and messages in the system.

## Notifications

Notifications are short system-generated updates that inform users about important workflow events, such as application actions or recruitment decisions.

## Messages

Messages support direct communication between users, especially between TA applicants and Module Organisers.

## Why the distinction matters

The two features serve different purposes:

- notifications highlight events
- messages support conversation

## Practical value

Together, notifications and messages make the recruitment workflow feel more complete and easier to follow from both the applicant and reviewer side.
