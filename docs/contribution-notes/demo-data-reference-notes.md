# Demo Data Reference Notes

## Purpose

This note summarizes the role of demo data in the TA Recruitment System and provides a quick reference for demonstration preparation.

## Why demo data matters

The sample dataset makes it possible to demonstrate realistic workflows without manually building every account, profile, job, and application before each run.

## Demo account categories

The demo dataset includes:

- TA accounts
- MO accounts
- Admin account

## Workflow support

The demo data supports:

- applicant-side profile and application flow
- MO-side review and decision flow
- Admin-side monitoring and oversight flow

## Practical value

This makes the final system easier to test, easier to explain, and more suitable for a live viva or acceptance-style demonstration.
