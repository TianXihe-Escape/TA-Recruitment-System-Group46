# Workload Monitoring Notes

## Purpose

This note explains the purpose of workload monitoring in the TA Recruitment System.

## Why workload monitoring matters

Recruitment decisions should not only consider whether an applicant is suitable, but also whether their accepted workload remains reasonable.

## Main idea

The system monitors accepted assignments so that workload visibility is available when reviewing hiring outcomes.

## What workload data can support

Workload-related information helps with:

- identifying potentially overloaded TAs
- comparing weekly and one-off work
- supporting admin review of accepted assignments

## Relationship to recruitment decisions

Workload monitoring adds an extra layer of oversight after application review, helping the system connect recruitment outcomes with operational fairness and feasibility.
