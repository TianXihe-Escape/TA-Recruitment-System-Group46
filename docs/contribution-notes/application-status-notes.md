# Application Status Notes

## Purpose

This note summarizes the meaning of the main application statuses used in the recruitment workflow.

## Common statuses

### `SUBMITTED`

The applicant has sent the application and it is waiting for review.

### `SHORTLISTED`

The applicant has passed the initial review stage and remains under active consideration.

### `INTERVIEW_INVITED`

The applicant has been selected to move to an interview-related stage.

### `ACCEPTED`

The applicant has been selected for the role, and the outcome can contribute to workload assignment.

### `REJECTED`

The application has been reviewed and not selected.

### `WITHDRAWN`

The applicant has chosen to withdraw the application before final completion of the process.

## Why status clarity matters

Clear status meanings help both applicants and reviewers understand the current state of each recruitment record.
